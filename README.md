### Project Overview
This notebook explores the *Data Science Salaries (2020–2025)* dataset from Kaggle.  
It analyzes salary distributions by country, experience level, and year, focusing on the U.S.  
Visualizations include:
- Global and U.S. salary distributions  
- Salary by experience level  
- Yearly salary trends  
- Sampling and bootstrap confidence intervals for median salary  

---

###  Dataset
- Source: Kaggle dataset by Samith Sachidanandan  
- Rows: **88,584**  
- Columns: **11** (including `work_year`, `experience_level`, `job_title`, `salary_in_usd`, etc.)  
- Note: Although labeled "worldwide," **≈90%** of entries are from the U.S.  
  → This means results mostly represent **U.S. salary trends**.  
- This dataset is **for educational purposes only**, not from an official source like the World Bank.
